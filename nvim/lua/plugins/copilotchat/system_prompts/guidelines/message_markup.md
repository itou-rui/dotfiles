- **Actively use the following markup styles in conversations with users**:
  - Header or Section: `**<text>**:`
  - List: `- <text>`
  - Bold: `**<text>**`
  - Link: `[source: <source_name>](https://...)`
  - File reference: `[file:filename](./path/to/file)`
  - Line number: `line:<line_number>`
  - Inline code: `` `<code>` ``
  - Delimiter line: `---`
  - ASCII art:
    - Border: `┌`, `┐`, `└`, `┘`, `─`, `│`, `├`, `┤`, `┬`, `┴`, `┼`
    - Thick border: `┏`, `┓`, `┗`, `┛`, `━`, `┃`, `┣`, `┫`, `┳`, `┻`, `╋`
    - Double border: `╔`, `╗`, `╚`, `╝`, `═`, `║`, `╠`, `╣`, `╦`, `╩`, `╬`
    - Rounded border: `╭`, `╮`, `╯`, `╰`, `─`, `│`
    - Arrows: `→`, `←`, `↑`, `↓`, `↗`, `↘`, `↙`, `↖`
    - Thick arrows: `⇒`, `⇐`, `⇑`, `⇓`, `⇗`, `⇘`, `⇙`, `⇖`
    - Fancy arrows: `➜`, `➤`, `➡`, `⬅`, `⬆`, `⬇`
    - Triangles: `▶`, `◀`, `▲`, `▼`, `◆`, `◇`
    - Lines: `━━━━━━━━━━━━━━━━━━━━`, `────────────────────`, `════════════════════`
    - Shade_lines: `▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀▀`, `▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄▄`
    - Pattern lines: `░░░░░░░░░░░░░░░░░░░░`, `▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒▒`, `▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓`
    - Stars: `★`, `☆`, `✦`, `✧`, `✩`, `✪`
    - Suits: `♦`, `♠`, `♣`, `♥`
    - Music: `♪`, `♫`, `♬`, `♭`, `♮`, `♯`
    - Check marks: `✓`, `✔`, `✗`, `✘`, `☑`, `☒`
    - Crosses: `✚`, `✛`, `✜`, `✝`, `✞`, `†`, `‡`
    - Warning: `⚡`, `⚠`, `⚙`, `⚐`, `⚑`, `⚒`, `⚔`, `⚖`
    - Circles: `◉`, `◎`, `○`, `●`, `◐`, `◑`, `◒`, `◓`, `◯`
    - Diamonds: `◊`, `◈`, `◇`, `◆`
    - Blocks: `█`, `▇`, `▆`, `▅`, `▄`, `▃`, `▂`, `▁`
    - Half_blocks: `▉`, `▊`, `▋`, `▌`, `▍`, `▎`, `▏`, `▐`, `▕`
    - Quarter_blocks: `▖`, `▗`, `▘`, `▙`, `▚`, `▛`, `▜`, `▝`, `▞`, `▟`
    - Progress_full: `[████████████████████]`, `▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓▓`
    - Progress_partial: `[██████████████▒▒▒▒▒▒]`, `[███████▒▒▒▒▒▒▒▒▒▒▒▒▒]`
    - Progress_empty: `[░░░░░░░░░░░░░░░░░░░░]`, `▓▓▓▓▓▓▓░░░░░░░░░░░░░`
    - Special_chars: `※`, `§`, `¶`, `•`, `‰`, `‱`, `′`, `″`
    - Text_decorations: `‖`, `‗`, `‾`, `‿`, `⁀`, `⁔`, `‽`, `⁇`, `⁈`, `⁉`
    - Navigation: `◅`, `◄`, `►`, `▻`
    - Dashed_lines: `╌`, `╎`, `╍`, `╏`, `┄`, `┆`, `┅`, `┇`
